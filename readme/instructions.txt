Главное меню:
при запуске программы доступны базовые файлы, все функции с ними поддерживаются
во время поиска программа может не отвечать, это нормально, нужно немного подождать
Плеер:
сначала галочками выделите необходимые файлы, потом нажмите кнопку load, а потом уже кнопку play. Чтобы загрузить новую очередь, также нажмите load после выбора галочек
Редактор:
выбранный файл при обрезке или изменении скорости сам не изменяется, создается новый с именем {старое название} _cutted, _slowed, _fastered
если вы моральный урод и решели что будет отличной идеей удалить редактируемый файл, то с наибольшей вероятностью программа не сломается, а вылезет окно с информацией о том, что файл поврежден
Общее:
программа в некоторых моментах может подвисать, в частности поиск и изменение скорости. Не надо принудительно ее закрывать
из-за особенностей виджета вывода треков, обои разрывает именно на этом виджете, если не нравится - сидите без обоев
